# Kestral Desktop Releases

This public repository contains signed Kestral desktop installers and auto-update metadata. The application source code is maintained separately in a private repository.

Release artifacts are generated and published by Kestral's desktop deployment pipeline. Do not commit application source code or credentials here.

For the latest stable installer, see [Releases](https://github.com/Kestral-Team/kestral-desktop-releases/releases/latest).
